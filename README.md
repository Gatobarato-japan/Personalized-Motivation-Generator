# Personalized Motivation Generator

Aplicación web interactiva que genera mensajes de motivación personalizados basados en el nombre ingresado por el usuario.

## Concepto
El proyecto busca crear una experiencia simple pero significativa, donde el usuario recibe palabras de aliento personalizadas al ingresar su nombre.

## Features
- Entrada de nombre del usuario
- Generación dinámica de mensajes personalizados
- Interfaz simple e intuitiva
- Respuestas motivacionales en tiempo real

## Tecnologías aplicadas
- HTML5
- CSS3
- JavaScript

## Objetivo
Explorar la manipulación del DOM y la creación de experiencias personalizadas en aplicaciones web.

## Estructura del proyecto
- index.html → estructura principal
- style.css → diseño visual
- script.js → lógica de generación de mensajes
- music.mp3 → música ambiental

## Ejemplo
Usuario: "Steve"  
Mensaje: "Steve, tienes la capacidad de lograr grandes cosas. Sigue adelante."
